## gRESTful
gRESTful helps you integrate RESTful web services with your Glib2 code.

### Dependencies
* python 2.7.x
* pygobject3
* pycurl

### Features
* Asynchronous to the bone.
* Access RESTful resources just as if they were GObject-based objects.
* Extremely easy to use.
